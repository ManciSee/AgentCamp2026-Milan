# AgentCamp 2026 Milan

Welcome to the official repository for **AgentCamp 2026 Milan** - a free community event focused on AI agents, tools, and practical applications.

## About AgentCamp

AgentCamp is a community-driven AI learning experience where developers and AI enthusiasts come together to explore the latest in AI agents and cutting-edge AI technologies. Our goal is to help everyone learn, share knowledge, and build real-world solutions with AI.

## Event Details

**Date:** Tuesday, March 3, 2026  
**Time:** 2:30 PM - 5:30 PM (W. Europe Standard Time)  
**Format:** Online & In-Person (Reggio Emilia, Italy)

### Sessions

#### 1. Building Intelligent Agents with GitHub Copilot, Agent Builder, and Microsoft Foundry

*Speakers: Anna Fabris, Matteo Conti (delivered in Italian)*

Learn how to design, evaluate, and operationalize AI agents using GitHub Copilot, Agent Builder, and Microsoft Foundry. This practical walkthrough covers model selection, system prompt crafting, tool integration, and agent evaluation.

#### 2. Developing Multimodal Agents: A Hands-On Workshop with Microsoft Foundry & AI Toolkit

*Speakers: Anna Fabris, Sergio Mancini (delivered in Italian)*

A practical overview of key technologies for developing modern AI solutions. Learn how to use AI Toolkit for VS Code, explore Microsoft Foundry, and discover the Model Context Protocol (MCP) for integrating agents with external data and tools. We'll work with Python, PostgreSQL, and pgvector for vector search capabilities.

#### 3. Orchestrating a Multi-Agent Architecture with Copilot Studio

*Speakers: Vittoria Marano, Sergio Mancini (delivered in Italian)*

Explore how Copilot Studio can orchestrate multiple intelligent agents built across different Microsoft platforms, turning distributed AI capabilities into one cohesive and intelligent solution.

#### 4. Building Agentic AI Systems with Microsoft Agent Framework in .NET

*Speaker: Andrea Beghini (delivered in Italian)*

A deep dive into Microsoft Agent Framework, an open-source SDK in .NET and Python, to build, deploy, and manage sophisticated multi-agent systems. It unifies the enterprise-ready foundations of Semantic Kernel with the innovative orchestration of AutoGen.
